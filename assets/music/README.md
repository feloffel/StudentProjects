# Hintergrundmusik

Lege hier deine MP3 ab, z.B. `hintergrund.mp3`.

Der Player auf der Seite ist auf `assets/music/hintergrund.mp3` verlinkt.
Anderen Dateinamen/Titel? In `index.html` beim `<audio id="musicAudio" src="...">`
den Pfad ändern und ggf. den Text in `<span id="musicTitle">` anpassen.

Hinweis: Browser starten Audio erst nach einem Klick (kein Autoplay) — daher der
Play-Button im Player.
