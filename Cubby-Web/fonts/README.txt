CUBBY – Schriften / Fonts
=========================

Die Hauptschrift "Mango" ist kommerziell und liegt nicht im Projekt.
Lege die Mango-Schriftdatei(en) HIER in diesen Ordner und benenne sie so:

    Mango-Regular.woff2   (bevorzugt, kleinste Datei)
    Mango-Regular.woff    (optional)
    Mango-Regular.otf     (optional)
    Mango-Regular.ttf     (optional)

Es reicht EINE der Dateien – die CSS lädt automatisch die erste verfügbare.
Sobald die Datei hier liegt, erscheinen alle verspielten Überschriften
(Logo-Schriftzug, große Headlines) in Mango.

Solange keine Mango-Datei vorhanden ist, wird automatisch "Baloo 2"
(runde, freundliche Google-Schrift) als optisch nahe Ersatzschrift genutzt,
damit nichts kaputt aussieht.

Poppins (Fließtext + normale Überschriften) wird von Google Fonts geladen.
